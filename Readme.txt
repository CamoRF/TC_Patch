Mod that aims to fix additional game issues that Dash Faction cannot due to needing level modifications or sophisticated .tbl edits.

\Changelog v0.1/
*Fix all geometry holes in levels l1s1, l4s4, l12s1, l5s2, and l4s3.
*Fix grenades sticking to surfaces using wtr_ material textures.
*Implement use key patch for medlabs stealth section by Goober.
*Implement fixed envirosuit pickup model by HeyitsDuke.
*Fix misaligned bathroom floor texture in level l6s3.
*Implement missing resources error fix by HeyitsDuke.
*Fix abrupt fog transition from level l1s3 to l2s1.
*Fix level l1s1 not using seamless floor texture.
*Implement Foley and entity fixes by HeyitsDuke.
*Fix missing lights in levels l5s3 and l5s4.
*Fix misaligned wall texture in level l6s2.
*Add collision to Capek cane projectiles.
*Fix jeep and submarine headlight glares.
*Fix missing door sounds in level l13s3.
*Add life value to vegetation objects.
*Restore missing lava in level l1s1.
*Fix missing lights in level l12s1.
*Better mouse input for vehicles.
*Add collision to all particles.
*Instant weapon swapping.

\Changelog v0.2/
*Fix Gryphon dying when skipping cutscene in levels l7s1 and l7s2.
*Add missing glass bullet impact sounds to weapons.tbl.
*Improve watermist and waterdrop alpha fade in.
*Use collisions for body bags.

\Usage/
Move TC_Patch_v0.1 to the mods folder of your RF installation and start game with mod using Dash Faction.
